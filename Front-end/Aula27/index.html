O que é o DOM?

O DOM (Document Object Model) é uma interface de programação para documentos HTML e XML. Ele representa a estrutura do documento como uma árvore de objetos, onde cada nó é um elemento ou um atributo do documento. Com o DOM, você pode manipular a estrutura, estilo e conteúdo dos documentos.

Estrutura Básica do DOM

  1. Document: O nó raiz que representa o documento inteiro.
  2. Element: Representa um elemento HTML, como ("div") ("p"), ou ("span").
  3. Text: Representa o texto dentro de um elemento.
  4. Attribute: Representa os atributos dos elementos, como id ou class.
  5. Manipulando o DOM com JavaScript

Aqui estão algumas operações básicas que você pode realizar com o DOM usando JavaScript:

  • Selecionar Elementos: Usando métodos como getElementById, querySelector, etc.
  • Alterar o Conteúdo: Usando propriedades como innerHTML ou textContent.
  • Modificar Estilos: Usando a propriedade style.
  • Adicionar/Remover Elementos: Usando métodos como appendChild, removeChild, etc.
