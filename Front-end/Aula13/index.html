CSS LOCAL (inline)

O CSS local é utilizado através do STYLE. Quando usando ele tem prioridade máxima no projeto.

CSS LOCAL("style")

O CSS LOCAL é usando a tag STYLE deve ser feito dentro das tags HEAD. Esse método tem a segunda prioridade quando usado todos os métodos de CSS no mesmo projeto

CSS EXTERNO("link")

O CSS externo é usado principalmente quando um projeto tem muitas páginas e todas tem foratações comum. Para usar o css usamos a tag link também dentro do HEAD.
